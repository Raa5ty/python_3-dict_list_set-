list:
    1. list.append(x) - Добавляет элемент в конец списка
    2. list.insert(i, x) - Вставляет на i-ый элемент значение x
    3. list.remove(x) - Удаляет первый элемент в списке, имеющий значение x. ValueError, если такого элемента не существует
    4. list.count(x) - 	Возвращает количество элементов со значением x
    5. list.sort([key=функция]) - Сортирует список на основе функции
str:
    1. len(S) - Длина строки
    2. S.replace(шаблон, замена[, maxcount]) - Замена шаблона
    3. S.split(символ) - Разбиение строки по разделителю
    4. S.upper()\S.lower() - Преобразование строки к верхнему\нижнему регистру
    5. S.strip([chars]) - Удаление пробельных символов в начале и в конце строки
set:
    1. len(s)
    2. set.add(elem) - добавляет элемент в множество
    3. set.remove(elem) - удаляет элемент из множества. KeyError, если такого элемента не существует
    4. set.discard(elem) - удаляет элемент, если он находится в множестве.
    5. set.copy() - копия множества.
dict:
    1. dict.clear() - очищает словарь.
    2. dict.copy() - возвращает копию словаря.
    3. dict.get(key[, default]) - возвращает значение ключа, но если его нет, не бросает исключение, а возвращает default (по умолчанию None).
    4. dict.items() - возвращает пары (ключ, значение).
    5. dict.keys() - возвращает ключи в словаре.
    6. dict.values() - возвращает значения в словаре.
